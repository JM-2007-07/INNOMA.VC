# INNOMA.VC

Современный адаптивный веб-сайт для цифрового продукта / IT-компании с акцентом на минимализм, удобство и современный UI/UX.

##  О проекте

INNOMA — это стильный и современный лендинг, созданный для презентации технологического продукта, сервиса или компании.  
Дизайн выполнен в минималистичном стиле с акцентом на:

- чистую типографику
- современную визуальную иерархию
- удобную навигацию
- адаптивность под мобильные устройства
- плавные UI-анимации
- удобный пользовательский опыт
